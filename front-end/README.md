# Kaya Frontend

Vue 3 staff dashboard for Kaya Reservation System.

## Scripts

```bash
npm run dev
npm run build
npm run test:unit
```

## Main pages

- Dashboard
- Reservations
- Customers
- Tables
- Closures
- Settings

The frontend consumes the backend settings endpoint so seating areas, timezone-adjacent defaults, and weekly hours come from the API instead of being hardcoded in page components.

See the root README for full application setup.
